# Bank Management System (Python)

A simple and structured Bank Management System built using Python and Object-Oriented Programming (OOP). This project demonstrates how basic banking operations can be implemented using clean and modular code.

---

## About the Project

This project is designed to practice important programming concepts such as Encapsulation, Data Validation, and Secure Operations in a real-world scenario. It allows users to create accounts, perform transactions, and track their transaction history.

It is suitable for beginners as well as students working on academic projects.

---

## Features

* Create a new bank account
* Deposit money
* Withdraw money
* Check account balance
* View transaction history
* Secure login using PIN
* Input validation for safe operations

---

## Concepts Used

* Object-Oriented Programming (OOP)
* Encapsulation (use of private variables)
* Data validation
* Modular programming (multiple files)
* Real-world logic implementation

---

## Project Structure

```
bank_system/
│
├── account.py        # Handles account operations
├── transaction.py    # Stores transaction details
├── bank.py           # Manages all accounts
└── main.py           # Entry point of the program
```

---

## How to Run the Project

1. Clone or download this repository
2. Open the project folder in your code editor
3. Run the following command:

```bash
python main.py
```

---

## How It Works

* Each user creates an account using a unique account number and a 4-digit PIN
* All operations require PIN authentication
* Transactions are recorded and stored during runtime
* Users can view their complete transaction history

---

## Security Features

* PIN-based authentication
* Private variables to protect sensitive data
* Validation checks for transactions
